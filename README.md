# HUAWEI WEAR ENGINE - FLUTTER

Huawei Wear Engine for Flutter, this plugin supports the following functions:
    - hasAvailableDevices
    - checkPermission (only one permission)
    - checkPermissions (permission array)
    - requestPermission
    - getBondedDevices
    - isAppInstalled
    - getAppVersion
    - ping
    - send (only text)
